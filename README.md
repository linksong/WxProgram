与慕课网的主要更新**区别**：<br/>
1.部分命名更改：后端见数据库：area.sql，application.properties，前端查看返回的json <br/>
2.删除如datasource和sessionFactory类、mybatis.xml的构造，**直接在SpringBoot中配置** <br/>
3.修改controller访问地址和访问方式，尽可能**符合 RESTful 规范**）<br/>
4.使用lombok，使代码更加简洁 <br/>
